氣體的性質與通性

氣體的通性

  • 壓縮性 (Compressibility)：受壓體積大幅縮小。
  • 膨脹性 (Expansibility)：充滿整個容器空間。
  • 擴散 (Diffusion)：氣體分子混入另一介質。
  • 逸散 (Effusion)：由小孔逃逸至真空。

溫度與平均動能

升高溫度，平均動能增加，分子運動速率增快。

$E_k = \frac{1}{2}mv^2 = \frac{3}{2}KT$

氣體壓力的定義

壓力的定義 (Pressure)

單位面積上所受的垂直作用力。

$P = \frac{F}{A} = \frac{m \times a}{A}$

流體壓力公式

液體產生的壓力與深度及密度有關。

$P = h \times d$

氣體壓力的導証

密閉容器壓力因氣體分子快速運動碰撞器壁產生的衝量而形成：

1. 動量 (Momentum)：

$p = mv$

2. 作用力 (Force)：

$F = ma = \frac{mv' - mv}{t}$

* 壓力即為單位時間、單位面積內，大量分子碰撞器壁之動量變化量的總和。

開口式水銀壓力計

(A) $P_{\text{gas}} > P_{\text{atm}}$

$P_{\text{gas}} = P_{\text{atm}} + h$

(B) $P_{\text{gas}} < P_{\text{atm}}$

$P_{\text{gas}} = P_{\text{atm}} - h$

閉口式水銀壓力計

原理

右端抽成 真空 (Vacuum)，不計大氣壓力。

此時水銀柱高度差 $h$ 即代表氣體壓力。

$P_{\text{gas}} = h \text{ mmHg}$

注意：不論外界大氣壓如何改變，閉口式壓力計的 $h$ 僅隨氣體壓力改變。

綜合練習

解答：910 mm
$P_{\text{gas}} = P_{\text{atm}} + h = 760 + 150 = 910 \text{ mmHg}$
改用閉口式，$P_{\text{gas}} = h' = 910 \text{ mm}$

例題 1

將氣體緩緩加入連接開口壓力計的容器中，直到右管水銀面較左管高 $150 \text{ mm}$。若一大氣壓為 $760 \text{ mmHg}$，改用 閉口式 壓力計，則高度差為何？

(開口式)
(閉口式)
解答：1.5
$P_A = h$, $P_B = 1.5h$
$P_B / P_A = 1.5$

例題 2

某閉口式壓力計充入 A 氣體後，高度差為 $h$。換成 B 氣體後，高度差變為 $1.5h$。則 $P_B / P_A$ 之值為何？

閉口式(A)
閉口式(B)

綜合練習 (3)

解答：(A)
$P_A = h_R - h_L = 3h_L - h_L = 2h_L = 0.5C$
$P_B = h_R - h_L = 4h_L - h_L = 3h_L = 0.6C$
$P_B / P_A = 0.6 / 0.5 = 1.2$
