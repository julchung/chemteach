1897J.J. Thomson

陰極射線的特性

  • ✦ 由陰極射出（管壓 $10^{-6}$ atm），無場時直線前進
  • ✦ 為帶電子流（非光線），帶負電
  • ✦ 受電場影響：偏向正極板
  • ✦ 受磁場影響：依安培右手定則偏折
  • ✦ 撞擊 ZnS 產生螢光效應
  • ✦ 可使氣體電離、底片感光

湯木生 — 質譜儀與同位素 (1913)

1913質譜儀 (Mass Spectrometer)

工作原理（四步驟）

  1. 氣體被電子束撞擊 → 形成帶電陽離子
  2. 陽離子受電場加速進入真空室
  3. 受磁場影響而偏折，曲率半徑 $r \propto \sqrt{m/q}$
  4. 打在底片上，位置反映質量，深淺反映含量
$r = \dfrac{\sqrt{2mV/q}}{B}$（曲率半徑公式）
同位素：原子序相同、中子數不同 → 同一元素、不同質量數
例：$^{20}$Ne(90.9%)、$^{21}$Ne(0.26%)、$^{22}$Ne(8.8%)

離子偏折軌跡動畫

🔵 輕離子（偏折多，r 小） |  🔴 重離子（偏折少，r 大）
底片位置差 → 質量差 → 同位素鑑定

範例練習 1 — 質譜儀荷質比計算

📝 解析步驟

① 已知：$X^{2+}$ 之荷質比 $= \dfrac{2e}{m} = 4820$ 庫侖/克
② 代入電量：$\dfrac{2 \times 1.6\times10^{-19}}{m} = 4820$
③ 解得：$m = \dfrac{3.2\times10^{-19}}{4820} \approx 6.64\times10^{-23}$ 克
④ 換算原子量：$\dfrac{6.64\times10^{-23}}{1.67\times10^{-24}} \approx \mathbf{40}$（即鈣 Ca）
✔ 答案：(B) 40

範例練習 2 — 質譜儀同位素分析

📝 解析步驟

① $M_2^+$ 峰強度比源自同位素結合機率：設兩種同位素在自然界含量為 $p, q$。
② 雙原子分子機率分佈為 $p^2 : 2pq : q^2 = 16 : 8 : 1$
③ 開根號可得 $p : q = 4 : 1$。因此該元素有 2 種同位素。
④ 單原子陽離子 $M^+$ 的強度比即為含量比，故 $a:b = p:q = \mathbf{4:1}$。
⑤ 平均原子量計算：$m_a \times \frac{4}{5} + m_b \times \frac{1}{5} = 24.4$
⑥ 代入 $m_b = 26$：$0.8 m_a + 5.2 = 24.4 \Rightarrow 0.8 m_a = 19.2 \Rightarrow m_a = \mathbf{24}$。
✔ 答案：(A) 2 種   (B) 4:1   (C) 質量數 24

密立坎 (R.A. Millikan) — 油滴實驗 (1909)

1909R.A. Millikan

實驗設計

霧化油滴通過帶電極板，調整電場使油滴靜止懸浮：

$qE = mg \;\Rightarrow\; q = \dfrac{mg}{E}$

實驗結論

測量多顆油滴電量，取最大公因數即為基本電荷：

$e = 1.6\times10^{-19}$ 庫侖 (C)
結合湯木生荷質比 → 電子質量：
$m_e = \dfrac{1.6\times10^{-19}}{1.76\times10^{8}} \approx \mathbf{9.1\times10^{-28}}$ 克

油滴實驗互動動畫

調整電場強度：
電場適中 → 油滴靜止懸浮

範例練習 3 — 油滴實驗電子數計算

📝 解析步驟

① 觀察到五顆油滴電量：3.2、4.8、6.4、8.0、9.6（e.s.u.）
② 求最大公因數：$\gcd(3.2,4.8,6.4,8.0,9.6) = \mathbf{1.6}$ e.s.u. → 此為 1 個電子電量
③ 另一油滴電量 $= 24.0$ e.s.u.
④ 電子數 $= 24.0 \div 1.6 = \mathbf{15}$ 個
✔ 答案：(D) 15 個電子

1911E. Rutherford

實驗設計

以 α 粒子（$He^{2+}$）高速轟擊極薄金箔，四周螢光屏偵測。

實驗結果 → 結論

✅ 絕大多數穿透 → 原子大部分是空的

💥 極少數反彈 → 正電荷質量集中於原子核

體積比 ≈ $10^{15}$（原子核極小，質量卻佔幾乎全部）

莫斯利 (H. Moseley) — X射線與原子序 (1913)

1913H. Moseley

實驗設計

以高速電子撞擊不同金屬靶（陽極），測量其發出的特徵 X 射線頻率（$\nu$）。

實驗發現與結論

發現 X 射線頻率的平方根 $\sqrt{\nu}$ 與靶金屬的原子序（$Z$）成正比：

$\sqrt{\nu} = a(Z - b)$
✅ 證實：元素的化學性質決定於原子序（質子數），而非原子量。
✅ 貢獻：修正門得列夫週期表以原子量排列的缺失，重新定義元素。
✅ 預測：成功預測了當時尚未發現的元素（如鎝 Tc、鉕 Pm 等）。

⚛️ 原子核組成的確認

1919
拉塞福 — α 粒子撞擊氮核（$^{14}_{7}$N）：
$^{4}_{2}He + ^{14}_{7}N \rightarrow ^{17}_{8}O + ^{1}_{1}H$（質子）
1932
查兌克 — α 粒子撞擊鈹核（$^{9}_{4}$Be）：
$^{4}_{2}He + ^{9}_{4}Be \rightarrow ^{12}_{6}C + ^{1}_{0}n$（中子）
📋 元素表示法 $^{A}_{Z}X^{n\pm}$
基本定義： $Z$ = 質子數（原子序） ｜ $A$ = 質量數 ｜ $A-Z$ = 中子數 陽離子 ($n+$)： 失去 $n$ 個電子 $\rightarrow$ 電子數 $= Z - n$ 陰離子 ($n-$)： 得到 $n$ 個電子 $\rightarrow$ 電子數 $= Z + n$

📝 解析

核反應中質量數、質子數守恆：
$^{4}_{2}He + ^{14}_{7}N \rightarrow ^{A}_{Z}O + ^{1}_{1}H$
質量數：$4+14 = A+1 \Rightarrow A=\mathbf{17}$
質子數：$2+7 = Z+1 \Rightarrow Z=\mathbf{8}$（氧）
✔ 答案：(A) $^{17}_{8}$O
