電子軌域的基本概念

古典模型 vs 量子力學

拉塞福與波耳模型： 電子繞著一定的圓形軌道運行（如同行星繞太陽）。這種觀念無法解釋多電子原子的行為。

量子力學模型：

  • 波動性與粒子性：高速運動的電子同時具有波動和粒子的特性（繞射現象）。
  • 測不準原理：電子在原子核附近的空間以三度空間快速運動，其確切軌跡無法測定。
  • 機率分佈：我們只能預測電子出現的機率空間。

「軌域」的定義 (Orbital)

軌域是指在原子核外空間中，電子出現機率最高（通常大於 90% ~ 100%）的區域形狀。

量子數：主量子數 ($n$)

數值：$n = 1, 2, 3, 4, ...$（正整數）

符號：K, L, M, N ...殼層

意義：決定電子的主要能量及電子與原子核間的平均距離。

  • $n$ 越大，軌域體積越大。
  • $n$ 越大，電子能量越高（離核越遠）。
  • 決定了電子所在的「主殼層」(Principal shell)。

量子數：角量子數 ($l$)

也稱軌域量子數，用以描述主殼層下的「副殼層」(Subshell)。

數值範圍：$l = 0, 1, 2, ..., (n-1)$

（對於主量子數 $n$，共有 $n$ 個可能的 $l$ 值）

意義：決定了軌域的空間形狀。

同一個主殼層中，$l$ 值越大，其副殼層的能階越高（即 $s < p < d < f$）。

軌域形狀代號對照

$l$ 值 代號 軌域形狀
0 s 軌域 球型 (Spherical)
1 p 軌域 啞鈴型 (Dumbbell)
2 d 軌域 花瓣型
3 f 軌域 複雜型

原子軌域形狀示意圖

s 軌域 (l=0) 球形
p 軌域 (l=1) 啞鈴形
d 軌域 (l=2) 花瓣與環形

量子數：磁量子數 ($m$)

數值範圍：$m = -l, ..., 0, ..., +l$

（對於給定的 $l$，共有 $2l + 1$ 個可能值）

意義：決定軌域在三維空間中的方向與個數。

註：在無外加磁場下，$m$ 值對軌域的大小、形狀、能量不影響。

  • 當 $l = 0$ (s軌域)：$m = 0$
    ➔ 只有 1 個方向（球狀對稱）
  • 當 $l = 1$ (p軌域)：$m = -1, 0, +1$
    ➔ 有 3 個方向 ($p_x, p_y, p_z$)
  • 當 $l = 2$ (d軌域)：$m = -2, -1, 0, +1, +2$
    ➔ 有 5 個方向
量子數與波函數 (The Orbitron)

軌域空間伸展方向 ($2l+1$) 示意圖

l=0 (s 軌域) ➔ 1 個方向 (m=0)
l=1 (p 軌域) ➔ 3 個方向 (m=-1, 0, +1)
l=2 (d 軌域) ➔ 5 個方向 (m=-2, -1, 0, +1, +2)

量子數：旋量子數 ($s$ 或 $m_s$)

數值：僅有 $+\frac{1}{2}$ 與 $-\frac{1}{2}$ 兩個可能值

意義：表示電子在空間中自轉的方向。

在量子力學中，電子除了在軌域中運動外，本身也帶有「自旋」的內在性質。

自旋會產生微小的磁場。根據包立不相容原理，同一個軌域內最多只能容納兩個電子，且它們的自旋方向必須相反，使得磁性互相抵消。

$s = +\frac{1}{2}$

自旋向上 ( $\uparrow$ )

$s = -\frac{1}{2}$

自旋向下 ( $\downarrow$ )

白色虛線與箭頭 (↑/↓)： 代表電子的自旋軸心與產生的磁極方向（自旋會使電子像一顆微小的電磁鐵，產生有方向性的微小磁場）。

黃色圓環與箭頭 (➔)： 代表電子的空間自轉方向（由自旋軸上方往下看：自旋向上為逆時針自轉，自旋向下為順時針自轉）。

極重要規律總結： 主量子數為 $n$，則有 $n$ 種副殼層； 同一主殼層中共有 $n^2$ 個軌域； 最多可容納 $2n^2$ 個電子！

電子軌域能階：單電子 vs 多電子對照

(一) 單電子原子 (類氫原子如 H)

軌域能量僅由主量子數 $n$ 決定，同主殼層的軌域為簡併態 (Degenerate)

能階高低：$1s < 2s = 2p < 3s = 3p = 3d < 4s = ...$

(二) 多電子原子系統

受同電性遮蔽 (Shielding) 影響，能量隨 $(n+l)$ 規則分裂與交錯

能階分裂：$1s < 2s < 2p < 3s < 3p < \mathbf{4s < 3d} < 4p ...$

範例練習 1

解析步驟

1. 軌域量子數 $l$ 的可能範圍為 $0, 1, 2, ..., (n-1)$。
2. 各選項分析：
    (A) 2s：$n=2$, $l=0$。($0 < 2$)，合理。
    (B) 4f：$n=4$, $l=3$。($3 < 4$)，合理。
    (C) 2d：$n=2$, $l=2$。($l$ 必須小於 $n$，故 $l=2$ 不可能存在)。
    (D) 7p：$n=7$, $l=1$。($1 < 7$)，合理。


答案：(C) 2d

不存在的軌域判斷

依量子力學，下列何種軌域不存在？

(A) 2s (B) 4f (C) 2d (D) 7p

範例練習 2

解析步驟

1. 判斷軌域符號對應的量子數：
    「3p」軌域代表主量子數 $n = 3$。
    「p」副殼層代表角量子數 $l = 1$。
2. 判斷磁量子數 $m$ 與旋量子數 $s$ 的合理範圍：
    $m$ 必須在 $-l$ 到 $+l$ 之間，因此 $m = -1, 0, +1$。
    $s$ 只能是 $+\frac{1}{2}$ 或 $-\frac{1}{2}$。
3. 檢查選項：
    (A) $l=2$ (錯，應為1)
    (B) $l=3$ (錯，應為1)
    (C) $l=2$ (錯，應為1)
    (D) $n=3$, $l=1$, $m=-1$, $s=+\frac{1}{2}$ (完全符合)


答案：(D)

量子數的合理性組合

對於一個 3p 軌域上的電子，其四個量子數 $(n, l, m, s)$，哪一組可能存在？

  • (A) $n=3$，$l=2$，$m=-1$，$s=+\frac{1}{2}$
  • (B) $n=3$，$l=3$，$m=0$，$s=-\frac{1}{2}$
  • (C) $n=3$，$l=2$，$m=+2$，$s=-\frac{1}{2}$
  • (D) $n=3$，$l=1$，$m=-1$，$s=+\frac{1}{2}$
